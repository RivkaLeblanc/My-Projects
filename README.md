# My Projects Repository

This repository contains three distinct full-stack projects, each built with different technology stacks. A comprehensive collection showcasing expertise across multiple frameworks and languages.

---

## 📁 Projects Overview

### 1. **TripManager - Angular**
A modern, single-page application built with Angular for managing and planning trips.

**Tech Stack:**
- Frontend: Angular (TypeScript)
- Build Tool: Angular CLI
- Styling: CSS/SCSS

**Features:**
- Responsive design
- Interactive trip management
- Real-time updates

**Location:** `TripManager - Angular/`

---

### 2. **SongApp - Node.js**
A full-stack web application for managing songs with user authentication and CRUD operations.

**Tech Stack:**
- Backend: Node.js with Express.js
- Database: MongoDB with Mongoose
- Frontend: EJS templating engine
- Authentication: bcrypt & JWT
- Other: cookie-parser, dotenv

**Features:**
- User registration and authentication
- Add, edit, and delete songs
- Song details page
- Secure session management

**Location:** `SongApp - Node.js/`

---

### 3. **SnowGood - React**
A dynamic React-based web application for interactive user experiences.

**Tech Stack:**
- Frontend: React
- Styling: CSS/SCSS
- State Management: React Hooks

**Features:**
- Component-based architecture
- Responsive UI
- Interactive features

**Location:** `SnowGood - React/`

### 4. **Java Project**
A Java application organized with Maven and packaged as a complete project folder.

**Tech Stack:**
- Language: Java
- Build Tool: Maven
- Data: H2 file
- Frontend: React + TypeScript

**Features:**
- Java application logic
- Project structure with backend, frontend, and data assets
- Easily buildable with Maven

**Location:** `RecyleSystem - Java/`
